Bases: capybara.driver.base.Base

A Capybara driver that uses Selenium WebDriver to drive a real browser.

Parameters:
  • app (object) – The WSGI-compliant app to drive.
  • browser (str, optional) – The name of the browser to use. Defaults to “firefox”.
  • desired_capabilities (Dict[str, str | bool], optional) – Desired capabilities of the underlying browser. Defaults to a set of reasonable defaults provided by Selenium.
